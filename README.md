# 🐍 Classic Snake Game (Pygame)

Uma implementação fiel do clássico jogo da "cobrinha", desenvolvida em Python com a biblioteca **Pygame**. O projeto foca em lógica de movimento em grid, detecção de colisão e gestão de estados simples.

## ✨ Destaques do Código
- **Movimento em Grid:** Lógica baseada em blocos de 20x20 pixels, garantindo o movimento clássico.
- **Detecção de Colisão:** Sistema que monitora bordas da tela e o próprio corpo da serpente.
- **Geração Aleatória:** A comida surge em coordenadas aleatórias, sempre alinhada ao grid do jogo.
- **Código Limpo:** Uso de constantes para facilitar o ajuste de dificuldade (velocidade) e visual (cores).

## 🚀 Como Executar
1. Instale o Pygame: `pip install pygame`
2. Execute o script: `python snake.py`

## 🕹️ Controles
- **Setas Teclado:** Movem a cobra.
- **Objetivo:** Coma o bloco vermelho para crescer e evite bater nas paredes ou em si mesma.
